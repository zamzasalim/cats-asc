# Cats-BOT Automation Script

## Prerequisites

- Node.js (v12.0.0 or higher recommended)
- npm (Node Package Manager)


### Auto Install 

```
git clone https://github.com/zamzasalim/cats-asc.git
```


```
cd Cats-BOT
```



```bash
npm install
```

### Edit data
```
nano data.txt
```

### Run
```
node main.js
```
